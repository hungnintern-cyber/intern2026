# デバッグチェックリスト

エラーやバグが発生した際の解決手順メモ：

1. **再現:**
   - 同じエラーが毎回発生するか確認する。
   - どの入力値（Input）でエラーになるか特定する。

2. **ログ確認:**
   - エラーメッセージを読み、ファイル名と行番号を特定する。
   - `SyntaxError`（構文）か `TypeError`（型）かを確認する。

3. **切り分け:**
   - 関係ないコードをコメントアウトして、問題の範囲を狭める。

4. **現状把握:**
   - エラー行の直前で `print(変数)` を使い、実際の値を確認する。
   - 変数のデータ型 `type(変数)` が想定通りか確認する。

5. **言語化:**
   - コードの処理ロジックを自分自身に説明し、論理的な矛盾を探す（ラバーダッキング法）。

6. **検証:**
   - 修正後、再度テストを実行して解決したか確認する。
